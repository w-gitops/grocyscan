# 20.10 User Stories

This document captures all user stories for Homebot, organized by feature area.

## 20.10.1 Core Scanning

| ID | As a... | I want to... | So that... | Priority |
|----|---------|--------------|------------|----------|
| US-001 | user | scan a barcode with my tablet camera | I can quickly add products | [CRITICAL] |
| US-002 | user | use my USB barcode scanner in kiosk mode | I can rapidly scan multiple items | [CRITICAL] |
| US-003 | user | have the system look up product details automatically | I don't have to type them | [CRITICAL] |
| US-004 | user | review product details before adding | I can verify accuracy | [CRITICAL] |
| US-005 | user | configure auto-add without review | I have faster workflow when confident | [CORE] |
| US-006 | user | add exactly 1 unit per scan | inventory is accurate | [CRITICAL] |

## Priority Legend

| Tag | Meaning |
|-----|---------|
| **[CRITICAL]** | Must be in MVP, hard to change later |
| **[CORE]** | Core MVP functionality |
| **[FLEX]** | Can be deferred without blocking |
| **[FUTURE]** | Explicitly deferred |
