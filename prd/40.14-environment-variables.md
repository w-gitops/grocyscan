# 40.14 Environment Variable Reference

## Complete Environment Variable List

```bash
# ============================================
# APPLICATION CORE
# ============================================
HOMEBOT_VERSION=2.0.0
HOMEBOT_ENV=production              # development, staging, production
HOMEBOT_DEBUG=false
HOMEBOT_HOST=0.0.0.0
HOMEBOT_PORT=3334
HOMEBOT_SECRET_KEY=                 # Required: 32+ character secret
HOMEBOT_BASE_URL=https://homebot.example.com  # For QR codes and links

# ============================================
# API DOCUMENTATION
# ============================================
DOCS_ENABLED=true                   # Enable /docs and /redoc
OPENAPI_URL=/api/v2/openapi.json

# ============================================
# DATABASE
# ============================================

DATABASE_POOL_SIZE=5

# ============================================

# ============================================

# ============================================
# MULTI-TENANT
# ============================================

# ============================================
# GROCY COMPATIBILITY
# ============================================
GROCY_COMPAT_ENABLED=true
GROCY_API_KEY_HEADER=GROCY-API-KEY

# ============================================
# LLM CONFIGURATION
# ============================================

# ============================================

# ============================================
# AUTHENTICATION
# ============================================

AUTH_JWT_ALGORITHM=HS256
AUTH_JWT_EXPIRY_HOURS=24
AUTH_REFRESH_EXPIRY_DAYS=7

# Service Tokens

# ============================================
```

## Development vs Production

### Development

```bash
HOMEBOT_ENV=development
HOMEBOT_DEBUG=true
DATABASE_ECHO=true
LOG_LEVEL=DEBUG
LOG_FORMAT=console
CORS_ORIGINS=http://localhost:9000,http://localhost:3334
SESSION_COOKIE_SECURE=false
```

### Production

```bash
HOMEBOT_ENV=production
HOMEBOT_DEBUG=false
DATABASE_ECHO=false
LOG_LEVEL=INFO
LOG_FORMAT=json
CORS_ORIGINS=https://homebot.example.com
SESSION_COOKIE_SECURE=true
```
