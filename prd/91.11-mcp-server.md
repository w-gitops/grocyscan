# 91.11 Model Context Protocol (MCP) Server

## 91.11.1 Overview

Homebot exposes an MCP server that enables AI assistants (Claude, Cursor, custom agents) to:
- Look up product information by barcode
- Add products to inventory
- Search products by name
- Manage locations and containers
- Track consumption and routines

## 91.11.2 MCP Server Implementation

    Add a product to inventory.
    
    Args:
        barcode: The barcode of the product to add
        location_id: Storage location UUID
        best_before: Expiration date in ISO format (YYYY-MM-DD)
        quantity: Number of units to add (default: 1)
    """
    pass


@mcp.tool()
async def search_products(query: str, limit: int = 10) -> list[dict]:
    """Search for products by name."""
    pass


@mcp.tool()

    """Get products expiring within the specified number of days."""
    pass


@mcp.tool()

    """List all storage locations."""
    pass


@mcp.tool()
async def log_consumption(
    product_id: str,

    """Log consumption of a product."""
    pass


@mcp.tool()

## 91.11.5 LangChain Integration

```python
from langchain_mcp_adapters.client import MultiServerMCPClient
from langchain_mcp_adapters.tools import load_mcp_tools

client = MultiServerMCPClient({
    "homebot": {
        "url": "http://localhost:3335",
        "transport": "streamable-http"
    }
})

async with client.session("homebot") as session:
    tools = await load_mcp_tools(session)
    result = await session.call_tool("lookup_barcode", {"barcode": "012345678901"})
```
